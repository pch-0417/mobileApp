## 연습 내용
- W1: 코틀린의 기본 Text, Row, Column, Button을 사용해봄.
- W3: 코틀린을 사용하여 이미지를 넣고 버튼을 만들어봄.
- W4: 자신만의 프로필 카드를 만들고 그에 맞는 TEXT문구 추가.
- W5: 타이머를 만들고 상태변수를 다루어봄.
- W6: 게임을 만들며 상태변수를 더 깊이있게 사용.
